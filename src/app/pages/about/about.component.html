AngularUniversity

Catálogo mundial de universidades desenvolvido com Angular

Objetivo do Projeto

O AngularUniversity é uma aplicação web responsiva que permite pesquisar e explorar universidades de qualquer país do mundo. O sistema consome a API pública Universities Hipolabs e oferece recursos como favoritos, histórico de pesquisas e filtragem local — tudo com uma interface moderna e adaptável para qualquer dispositivo.

Tecnologias Utilizadas

Angular 17
TypeScript
Bootstrap 5 + Bootstrap Icons
HttpClient (consumo REST)
Local Storage
SCSS

Funcionalidades

Pesquisa por país Filtro local por nome Favoritos Histórico Dashboard Ordenação A-Z Responsivo Dark/Light Mode Paginação

API Utilizada

FonteUniversities Hipolabs
AutenticaçãoNenhuma
FormatoJSON
CustoGratuita
http://universities.hipolabs.com/search?country=Brazil
